# AI_Research
The following project is on AI Research, this repository consists of Interview Transcripts, Data, Research Proposals, References and Scripts used for data processing. 

- Data -> Consists of raw and processed csv files
- Images -> Consists of png files
- Proposals -> Consists of proposal drafts (.docx)
- Interview Transcripts -> Survey Questionare and interview transcripts
- Scripts -> Python script used for data processing
- Paper Drafts -> Final Draft of the paper
- References -> PDF files of the references used

Guidelines:-
- Before running the script ensure the path is added properly
- Follow the same naming conventions and structure
- Ensure to redact participant names from the file to ensure user privacy
- While adding a reference please add the type and title
